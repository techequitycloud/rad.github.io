# About RAD Platform

**RAD Platform** is a hands-on Google Cloud certification training platform built by
**Tech Equity Cloud**. RAD stands for **Rapid Application Deployment** — not to be
confused with "Rapid Application Development" low-code tools.

Instead of slides or simulated sandboxes, RAD deploys pre-configured,
industry-realistic infrastructure ("modules") into your **own Google Cloud
project**, so every certification exam objective maps to a real, inspectable
cloud resource you deployed yourself.

## What the platform covers

- **159 module reference guides** — configuration references for deploying
  100+ open-source and cloud-native applications on Cloud Run and GKE Autopilot
- **111 hands-on labs** — guided deploy → verify → operate → observe → tear-down
  walkthroughs in your own GCP project
- **Certification lab maps and section guides** for seven Google Cloud
  certifications: Associate Cloud Engineer (ACE), Professional Cloud Architect
  (PCA), Professional Cloud Developer (PCD), Professional Cloud Database
  Engineer (PCDE), Professional Cloud Network Engineer (PCNE), Professional
  Cloud DevOps Engineer (PDE), and Professional Cloud Security Engineer (PSE)
- **Design principles** documenting the engineering standards behind every module
