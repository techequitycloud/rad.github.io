# Help & Support

The **Help** page is the central resource center for all users. It provides access to documentation, tutorials, and support channels.

## Features

### 1. Documentation Guides
Access role-specific guides directly within the application:
*   **User Guide:** General instructions for deploying and managing modules.
*   **Partner Guide:** Instructions for publishing modules and managing listings.
*   **Admin Guide:** Comprehensive documentation on platform configuration, user management, and billing settings.
*   **Agent Guide:** Tips for referring users and tracking revenue.
*   **Finance Guide:** Details on billing reports, invoices, and revenue tracking.
*   **Support Guide:** Information on how to get help.

*Note: Some guides are only visible if you have the corresponding user role.*

### 2. Audio Tutorials
Some guides include **Audio Tutorials** and **Workflow Walkthroughs**. Use the embedded audio player to listen to a guided tour of the features.

### 3. Support Request
*   **Contact Support:** Use the built-in form to send a message to the support team.
*   **Subject & Message:** Describe your issue clearly.
*   **Priority:** (Optional) Indicate the urgency of your request.

### 4. Send Message (Admin Only)
Administrators have a special "Send Message" tool on this page to broadcast notifications or direct messages to specific users or groups.

## How to Get Help

1.  Navigate to the **Help** page.
2.  **Browse Guides:** Click through the tabs (User Guide, Partner Guide, etc.) to find written documentation.
3.  **Contact Support:** If you can't find the answer, switch to the **Support** tab (or scroll to the form) and submit a ticket.
