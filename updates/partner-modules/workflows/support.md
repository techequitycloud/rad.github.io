# Support Workflows

This document details the workflows for **Support Operators** and **Helpdesk Staff**. The primary role of support is to assist users with access issues, account inquiries, and general platform guidance.

## User Assistance

### Resetting a User Password
1.  Search for the user in the **Users** section.
2.  Open their profile.
3.  Click **Passwords**.
4.  Select **Reset Password** or **Change Password**.
    *   **Reset**: Generates a new password or token and emails it to the user.
    *   **Change**: Allows you to manually set a new password (verify identity first!).
5.  If the user is **Blocked** due to invalid attempts, click **Unblock**.

### Unlocking a User
1.  Navigate to the User's profile.
2.  Check the status. If "Blocked", click the **Unblock** button.
3.  Confirm the action.

### Updating User Information
1.  If a user requests a profile update (e.g., address change, email change):
2.  Verify the user's identity according to your organization's policy.
3.  Go to **Profile** > **Edit**.
4.  Update the necessary fields and click **Save**.

## Ticket & Message Management

### Handling User Messages
Cyclos has an internal messaging system.
1.  Navigate to **Messages** or **Inbox**.
2.  View messages sent to the Support group.
3.  Click on a message to read and **Reply**.

### Managing Tickets (If Enabled)
1.  Navigate to **Helpdesk** or **Tickets**.
2.  View open tickets assigned to your group.
3.  Update ticket status (e.g., Open -> In Progress -> Resolved).
4.  Add internal notes or public replies to the user.

## System Alerts

### Monitoring Alerts
1.  Check the **Alerts** dashboard for system notifications.
2.  Look for "User Alerts" (e.g., suspicious activity, multiple failed logins).
3.  Investigate and take appropriate action.
