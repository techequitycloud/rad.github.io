# Topic guides

These pages map the RAD repo to common engineering topics. Every claim links
to a concrete file (Cloud Function, Terraform module, GitHub Actions workflow,
etc.) so it can be verified against the source.

Pages are grouped by what kind of question they answer. Each topic has ONE
primary page; related pages link to it rather than restate the content.

## Practices — *how teams work*

- [SRE](practices/sre.md) — resilience patterns, runtime hosting, testing.
- [DevSecOps](practices/devsecops.md) — identity, secrets, webhook
  signatures, security skills.
- [CI/CD](practices/cicd.md) — PR gates, application deployment, container
  builds.
- [FinOps](practices/finops.md) — credit lifecycle, cost APIs, multi-provider
  payments.
- [Platform Engineering](practices/platform_engineering.md) — developer
  portal, CLI, module catalogue.
- [GitOps & Infrastructure as Code](practices/gitops_iac.md) — Terraform-
  driven infra, PR-gated plan/apply.
