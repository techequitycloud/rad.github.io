# Agent Revenue

The **Agent Revenue** tab allows Finance users and Agents to view revenue generated by agents. This view focuses on the performance of agents and the revenue they bring in through their referrals.

## Features

*   **Filter by Agent:** Select specific agents to view their revenue data.
    *   If you are an **Agent**, your email is automatically selected.
    *   If you are a **Finance User** or **Admin**, you can select one or multiple agents from the dropdown list.
*   **Date Range Filtering:** Specify a start and end date to filter the revenue data.
*   **Revenue Summary:** View total transactions, total credit cost, and total revenue for the selected period.
*   **Detailed Table:**
    *   **Date:** The date of the transaction.
    *   **User Email:** The email of the user who performed the transaction (the referred user).
    *   **Module Name:** The name of the module that generated revenue.
    *   **Credit Cost:** The cost in credits.
    *   **Revenue:** The revenue amount in the configured currency.
*   **Export to CSV:** Download the filtered data as a CSV file for external analysis.

## How to Use

1.  Navigate to the **Billing** page and click on the **Agent Revenue** tab.
2.  **Select Agents:**
    *   Use the dropdown to select one or more agents.
    *   Use "Select All" to select all available agents.
    *   Use "Load More" if there are many agents.
3.  **Select Date Range:** Enter a **Start Date** and **End Date**.
4.  **Fetch Data:** Click the **Fetch Agent Revenue** button.
5.  **View Results:** The table will populate with revenue data. Summary statistics are displayed above the table.
6.  **Export:** Click **Export to CSV** to download the report.

## Notes

*   Data is paginated. Use the controls at the bottom of the table to navigate through pages.
*   The system caches query results for performance. If you suspect data is missing, try refreshing the page or changing the filter slightly to trigger a new fetch.
