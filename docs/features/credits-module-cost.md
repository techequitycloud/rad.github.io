# Module Costs

The **Module Costs** tab provides a breakdown of your credit usage by module. This helps you understand which deployments are consuming the most resources.

## Features

*   **Cost Aggregation:** Groups spending by Module Name.
*   **Deployment Count:** Shows how many active deployments you have for each module type.
*   **Total Cost:** The total credits spent on each module type within the selected period.

## How to Use

1.  Navigate to the **Credits** page and click on the **Module Costs** tab.
2.  The table displays a list of modules you have deployed.
3.  Review the **Total Cost** column to identify your most expensive services.
4.  This data can help you decide whether to optimize your usage or undeploy unused modules.
