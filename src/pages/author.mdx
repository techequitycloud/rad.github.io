# Dr Shiyghan Emmanuel Navti

Dr Shiyghan Emmanuel Navti writes and maintains all documentation on this
site — the certification guides, hands-on lab walkthroughs, and module
references of the **RAD Platform** by [Tech Equity Cloud](/about), and builds
the [RAD Console](https://radmodules.dev), the deployment portal behind these
docs.

Every guide follows the same working method: deploy the real infrastructure
into a live Google Cloud project first, then document what was deployed,
verified, and operated — so the study material maps to inspectable cloud
resources rather than slides.

## Google Cloud certifications

The author holds six active Google Cloud certifications:

| Certification | Level |
|---|---|
| [Associate Cloud Engineer (ACE)](https://cloud.google.com/learn/certification/cloud-engineer) | Associate |
| [Professional Cloud Architect (PCA)](https://cloud.google.com/learn/certification/cloud-architect) | Professional |
| [Professional Cloud Developer (PCD)](https://cloud.google.com/learn/certification/cloud-developer) | Professional |
| [Professional Cloud DevOps Engineer (PDE)](https://cloud.google.com/learn/certification/cloud-devops-engineer) | Professional |
| [Professional Cloud Security Engineer (PSE)](https://cloud.google.com/learn/certification/cloud-security-engineer) | Professional |
| [Cloud Digital Leader (CDL)](https://cloud.google.com/learn/certification/cloud-digital-leader) | Foundational |

Each certification links to its official Google Cloud page describing the
exam the credential is awarded for.
