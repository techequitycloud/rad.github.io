# User Management

This tutorial covers the essential administrative task of managing users on the RAD Platform. As an administrator, you have full control over user access.

## What You'll Learn

- How to view and manage users.
- How to update user roles.

### Prerequisites

- You have completed the [Initial Platform Setup](./initial-setup.md) tutorial.

### Estimated Time

- **15 minutes**

---

## Step 1: Navigate to the Users Page

User management is centralized on the **Users** page.

1.  From the main navigation menu, click on **"Users"**.
    ![Users Menu](/img/site/10-admin-users-menu.png)
2.  This page displays a list of all users on the platform, along with their roles and status.
    ![Users Table](/img/site/10.1-admin-users-table.png)

## Step 2: Update User Roles

Roles determine what a user can see and do on the platform.

1.  In the user list, find the user you want to modify and click the **"Edit"** button.
2.  In the user details modal, you can change a user's role.
    ![Update User Role](/img/site/10.1.1-admin-users-table-update.png)
3.  Click **"Save"** to apply the changes.

## Verification

-   Changes to user roles should be reflected immediately in the **Users** table.
