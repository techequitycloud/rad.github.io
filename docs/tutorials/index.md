# RAD Platform Tutorials

Welcome to the RAD Platform tutorials! These hands-on guides will help you learn the platform through practical, step-by-step examples. Whether you're a new user deploying your first application, a partner creating custom modules, or an administrator setting up the platform for your organization, these tutorials will guide you through real-world scenarios.

## How to Use These Tutorials

Each tutorial is designed to be self-contained and includes clear learning objectives, prerequisites, estimated completion time, step-by-step instructions with visual aids, verification steps to confirm success, and troubleshooting guidance for common issues.

We recommend following the tutorials in order within each category, as later tutorials often build on concepts introduced in earlier ones.

## Tutorial Categories

### User Tutorials

These tutorials are for users who want to deploy infrastructure using the RAD Platform. Start here if you're new to the platform.

**Getting Started**

The [Your First Deployment](./users/first-deployment.md) tutorial guides you through deploying your first application on the RAD Platform. You will learn how to sign in, select a module, configure your deployment, and monitor its progress. This is the perfect starting point for new users. The tutorial takes approximately 15 minutes to complete.

**Managing Resources**

The [Managing Your Credits and Subscriptions](./users/managing-credits.md) tutorial teaches you how to manage your billing and credits. You will learn how to subscribe to a plan, purchase additional credits, and track your spending. Understanding the credit system is essential for effective platform usage. This tutorial takes about 10 minutes.

**Troubleshooting**

The [Troubleshooting Failed Deployments](./users/troubleshooting-deployments.md) tutorial helps you diagnose and resolve deployment failures. You will learn how to read deployment logs, identify common error patterns, and fix typical issues. This is a crucial skill for successfully managing your infrastructure. Allow 20 minutes for this tutorial.

### Partner Tutorials

These tutorials are for partners who want to create and publish custom infrastructure modules. You need partner status to access these features.

**Getting Started with Module Development**

The [Publishing Your First Custom Module](./partners/first-module.md) tutorial walks you through the complete process of creating and publishing a custom module. You will learn how to set up your GitHub repository, structure your module correctly, and make it available for deployment. This foundational tutorial takes approximately 30 minutes.

**Building Quality Modules**

The [Creating a Production-Ready Module](./partners/production-module.md) tutorial covers best practices for building robust, maintainable modules. You will learn about proper variable definitions, comprehensive documentation, validation, and testing. This tutorial builds on the first module tutorial and takes about 45 minutes.

**Module Lifecycle Management**

The [Managing Module Versions and Updates](./partners/module-versions.md) tutorial teaches you how to manage the lifecycle of your published modules. You will learn about versioning strategies, handling updates, and communicating changes to users. This tutorial takes approximately 30 minutes and is essential for maintaining quality modules.

### Administrator Tutorials

These tutorials are for administrators who manage the RAD Platform for their organization. Administrator privileges are required.

**Initial Configuration**

The [Initial Platform Setup for Organizations](./administrators/initial-setup.md) tutorial guides you through the essential first-time configuration steps. You will learn how to set up global settings, configure the module repository, and prepare the platform for user onboarding. This comprehensive tutorial takes about 40 minutes.

**User Administration**

The [User and Credit Management](./administrators/user-management.md) tutorial covers ongoing user administration tasks. You will learn how to add users, assign roles, adjust credit balances, and use the Partner Credits feature. This tutorial takes approximately 25 minutes and is essential for day-to-day platform management.

### Advanced Tutorials

These tutorials cover advanced topics relevant to multiple user types. They provide deeper technical insights into the platform.

**Technical Deep Dive**

The [Understanding the RAD Platform Architecture](./advanced/platform-architecture.md) tutorial provides a comprehensive technical overview of how the platform works. You will learn about the integration with Google Cloud Platform services, the deployment pipeline, and the security model. This tutorial takes about 25 minutes and is valuable for advanced users and partners.

**Multi-Cloud Strategies**

The [Multi-Cloud Deployment Strategies](./advanced/multi-cloud-strategies.md) tutorial explores deploying infrastructure across multiple cloud providers. You will learn about the differences between AWS, Azure, and GCP, configuration considerations, and best practices for multi-cloud architectures. This tutorial takes approximately 30 minutes.

## Learning Paths

Depending on your role and goals, we recommend following these learning paths through the tutorials.

### New User Path

If you're new to the RAD Platform and want to start deploying infrastructure, follow this path. Begin with [Your First Deployment](./users/first-deployment.md) to learn the basics of deploying a module. Next, complete [Managing Your Credits and Subscriptions](./users/managing-credits.md) to understand the billing system. Then, work through [Troubleshooting Failed Deployments](./users/troubleshooting-deployments.md) to learn how to resolve issues. Finally, explore [Understanding the RAD Platform Architecture](./advanced/platform-architecture.md) for deeper technical knowledge.

### Partner Development Path

If you're a partner who wants to create custom modules, follow this path. Start with [Your First Deployment](./users/first-deployment.md) to understand the user experience. Then, complete [Publishing Your First Custom Module](./partners/first-module.md) to learn the basics of module creation. Next, work through [Creating a Production-Ready Module](./partners/production-module.md) to build quality modules. Follow with [Managing Module Versions and Updates](./partners/module-versions.md) to learn lifecycle management. Finally, study [Understanding the RAD Platform Architecture](./advanced/platform-architecture.md) to understand how your modules are executed.

### Administrator Path

If you're setting up and managing the platform for your organization, follow this path. Begin with [Initial Platform Setup for Organizations](./administrators/initial-setup.md) to configure the platform. Next, complete [User and Credit Management](./administrators/user-management.md) to learn user administration. Then, work through [Your First Deployment](./users/first-deployment.md) to understand the user experience. Follow with [Publishing Your First Custom Module](./partners/first-module.md) to understand module development. Finally, explore [Understanding the RAD Platform Architecture](./advanced/platform-architecture.md) for comprehensive technical knowledge.
